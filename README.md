## Projeto que retornar base64 de uma imagem

ao enviar uma requisição tipo POST para a url ela retorna o base64 desta imagem   


rodano servidor local

```bash
php artisan serve
```

url para teste  

metodo *POST*
```bash
http://localhost:8000/api/image
```
